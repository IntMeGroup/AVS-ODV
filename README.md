# AVS-ODV
This repository contains the AVS-ODV (Audio-Visual Saliency for Omnidirectional Videos) database.

## Database

The AVS-ODV database is available for download at the following link:

[Download AVS-ODV Database](https://drive.google.com/drive/folders/1jfqshH8p_itf7EWkHjvCPBfXpPR3jaED)
